ADOS Ground Station LCD UI mockups

480 × 320 SPI LCD (Waveshare 3.5" RPi LCD A) · resistive touch · top status bar + bottom tab bar

00 Chrome shell (empty)

Top status bar + bottom tab bar with empty content. Sets the chrome contract every page inherits.

01 Dashboard

Existing 4-tile dashboard refit into the new chrome. Tiles: Radio Link · Drone · Mesh · Uplink.

02 Video — link up

Live H.264 frame at 480×176 + permanent 56 px metrics strip (latency · RSSI · bitrate · FEC · FPS · channel).

03 Video — link down

Fail-soft when no stream is available. Centered hint card, metrics strip dimmed.

04 Settings — root

Flat 48 px scrollable list. Mix of toggle / enum / slider / text rows backed by the existing /api/v1/setup/apply REST surface.

05 Settings — channel picker (enum modal)

Enum picker modal with 48 px option rows. Used for Channel, Role, MCS, Topology, etc.

06 Settings — Wi-Fi password keyboard

On-screen keyboard for text fields. 4 rows × 12 keys at ~40×44. Resistive-touch-friendly key sizing.

07 Touch calibration wizard

First-boot 5-point tap. Stores 3×3 affine to /etc/ados/touch.calib. Skippable.

08 Tap feedback (active tab + inverse fill)

Single-frame inverse-fill on the touched tab. Makes resistive tap feel responsive even before the page swap renders.
